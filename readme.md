## Алгоритм работы

0. Установить необходимые зависимости:  
    > pip install -r requirements.txt
1. Создать `.env` файл с необходимыми переменными (есть [`.env.example`](./.env.example) для примера):
2. Выполнить [`prettify_df.ipynb`](./prettify_df.ipynb)  
    В результате выполнения **MULTI** вопросы станут **SINGLE**: варианты ответов уйдут в поле `answer`
3. Можно тестировать запросы. Для удобства создан [`test.ipynb`](./test.ipynb), в котором можно отследить выполнение каждого из этапов.
